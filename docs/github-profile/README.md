## Featured Final-Year Project

### Secure File Transfer Using Blockchain

[![Repository](https://img.shields.io/badge/View%20Repository-File--Sharing--Using--Blockchain-2ea44f?style=for-the-badge&logo=github)](https://github.com/Sauravtimilsina/File-Sharing-Using-Blockchain)
[![Release](https://img.shields.io/badge/Release-v1.0.0-blue?style=for-the-badge&logo=github)](https://github.com/Sauravtimilsina/File-Sharing-Using-Blockchain/releases/tag/v1.0.0)
[![License](https://img.shields.io/badge/License-MIT-yellow?style=for-the-badge)](https://github.com/Sauravtimilsina/File-Sharing-Using-Blockchain/blob/main/LICENSE)

A full-stack academic project for encrypted file sharing with OTP authentication, controlled access, Supabase storage, and SHA-256 integrity verification using a blockchain-style ledger.

## Project Highlights

- Encrypted file upload and download
- OTP-based account verification
- JWT-protected backend routes
- Password hashing with bcrypt
- SHA-256 file integrity verification
- Blockchain-style integrity records
- Controlled file sharing and revocation
- Supabase Postgres and private Supabase Storage
- Professional README, release notes, CI workflow, and security policy
